🚀 项目经历

基于Unitree G1人形机器人的具身导航Sim2Real部署 2025.10 - 2026.03
  • 工作目标：将开源具身导航模型从Isaac Sim仿真验证迁移至Unitree G1真机，完成端到端全流程部署
  • 仿真验证：主导搭建基于Isaac Sim的测试平台，支持场景导入、机器人初始化、指令输入、导航效果可视化
  • 真机部署：完成端云协同架构设计，通过ROS2订阅RGB-D与里程计数据，经Flask与服务端模型通信，由MPC控制器执行运动指令
  • 交互扩展：开发语音交互模块，支持多轮指令交互与实时语义中断
  • 工作效果：仿真端支持多场景快速验证；真机端以Unitree G1为载体，在多种典型室内场景下成功完成测试
基于零样本大语言模型决策的机器人具身导航算法框架 2025.03 - 2025.09
  • 工作目标：针对机器人指令遵循不准、易陷入循环的问题，设计并实现了基于大模型的任务规划与决策算法
  • 感知模块：设计Transformer路点预测与VLM语义理解方案，将视觉观测转化为文本描述，供大模型决策
  • 决策模块：设计目标导向的思维链推理机制，通过动作意图分解与动态进度评估提升指令遵循精度
  • 循环检测：设计基于位姿一致性的循环检测方案，结合CLIP语义相似度筛选路径，引导大模型重规划
  • 工作效果：导航成功率较最优基线提升21%，路径效率提升10%；以第一作者身份发表论文并被ICRA 2026接收，代码已开源至 GitHub
